# Task-1-Component-name-Card-Props-name-course-score-Task-2-Component-name-Form
App.jsx
Imports WelcomeMessage and Counter components.
Renders both inside a centered flexbox layout.
Defines simple inline styles for page layout.

Counter.jsx
Uses React’s useState to store count.
Displays current count and two buttons: Increment and Decrement.
Each button updates the state when clicked.
Has inline styles for card, buttons, and text.

WelcomeMessage.jsx
A simple header component.
Shows a title “Welcome to React Basics!” and a subtitle.
Uses inline styling for spacing and typography.

main.jsx
React entry point.
Loads <App /> inside the HTML element with id "root".
Uses ReactDOM.createRoot() for rendering in React 18.
